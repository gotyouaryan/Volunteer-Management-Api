# Volunteer Management API

A REST API for managing volunteers built with Node.js and Express.

## Features
- Add volunteers
- View all volunteers  
- Update volunteer status
- Delete volunteers

## API Endpoints
- GET /api/volunteers - Get all volunteers
- POST /api/volunteers - Add volunteer
- PUT /api/volunteers/:id - Update volunteer
- DELETE /api/volunteers/:id - Delete volunteer

## Tech Stack
- Node.js
- Express.js
- JSON file storage
